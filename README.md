This repository contains the Database Lab project for the Database Systems course at the University of Udine (UniUD), A.Y. 2025/2026.

## 📂 Project Structure

- [`report/`](report/): The official project report (written in Italian).
- [`schemas/`](schemas/): Entity-Relationship (ER) and logical diagrams.
- [`src/`](src/): Database source code, including the Dockerfile and SQL scripts.

## ⚠️ Repository Rules

**Important Notice:** Direct pushes to the `main` branch are not allowed. All code changes must be made exclusively through Pull Requests (PRs).
